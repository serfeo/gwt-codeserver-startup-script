Протестировано на Ubuntu 14.04, Google Chrome

0. Делаем mvn package для клиента
1. Запускаем скрипт - смотрим параметры [ в чат пример параметров скину ]
2. После того как codeserver запустится, открываем в Chrome http://127.0.0.1:9876
3. Перетаскиваем кнопки Dev Mode On и Dev Mode Off на панель быстрого досутпа ( закладки )
4. Запускаем клиент, логинимся
5. После загрузки клиента нажимаем закладку Dev Mode On
6. В открывшемся всплывающем окне нажимаем кнопку Compile
7. Ждем. Когда страница перезагрузится, открываем панель разработчиков вкладку Sources
8. Открываем сорцы для 127.0.0.1:9876, папка sourceMaps/...Module
9. Видим пакеты и java классы в них. В браузере работают brakepoints и watch.
10. Profit =)
